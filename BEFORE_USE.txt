⚠️ IMPORTANT (Windows security notice)

Windows may block files downloaded from the internet. To avoid issues, please follow the instructions below.

👉 IF YOU ARE USING THE INSTALLER (.exe):
- Right-click the downloaded installer file
- Select "Properties"
- Check the box: ✅ Unblock (if present)
- Click "Apply"
- Then run the installer

--------------------------------------------------

👉 IF YOU ARE USING THE ZIP VERSION:

BEFORE extracting the ZIP file:
1. Right-click the downloaded .zip file
2. Select "Properties"
3. At the bottom of the window, look for:
      "This file came from another computer and might be blocked"
4. Check the box: ✅ Unblock
5. Click "Apply"
6. THEN proceed with extraction

--------------------------------------------------

🔁 IF YOU HAVE ALREADY EXTRACTED THE FILES:

You must manually unblock the executable:

- Right-click "PMEG_Layout.exe" → Properties → check "Unblock" → Apply
- If needed, repeat the process for files inside the "_internal" folder

--------------------------------------------------

NOTE:
This step is required only once after downloading. It ensures the application runs correctly without Windows security restrictions.
