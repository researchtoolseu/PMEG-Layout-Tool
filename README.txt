PMEG Layout Tool – Quick Start

1. Open the PMEG Layout Tool from the desktop or Start Menu
2. Use the provided Excel template (pmeg_input.xlsm)
3. Fill in patient metadata and target vessels
4. Run the tool to generate:
   - PMEG Layout PDF
   - Film PDF
   - PNG image

Important:
- Ensure correct graft diameter and measurements
- Angles must be signed (right = positive, left = negative)
